# STM32 Register Programming

![STM32](https://img.shields.io/badge/MCU-STM32-blue)
![Language](https://img.shields.io/badge/Language-C-green)
![Level](https://img.shields.io/badge/Programming-Bare--Metal-orange)

Learning STM32 Bare-Metal Programming using direct register access.

## Topics

### Drivers
- GPIO
- RCC Clock
- Alternate Function I/O (AFIO)
- External Interrupt (EXTI)
- Timer
- ADC
- UART
- SPI
- I2C
  
## Development Board

- STM32F103C8T6 (Blue Pill)

## Goal

Understand STM32 peripherals by programming registers directly without using HAL libraries.
